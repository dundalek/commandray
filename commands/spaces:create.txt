Usage: heroku spaces:create

create a new space

 -o, --org ORG                 # organization to use
 -s, --space SPACE             # name of space to create
 --region REGION               # region name

Example:
  $ heroku spaces:create --space my-space --org my-org --region oregon
  Creating space my-space in organization my-org... done
